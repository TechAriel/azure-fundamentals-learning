# Creating an Azure Resource (Virtual Machine)

**Platform:** Microsoft Azure  
**Level:** Azure Fundamentals    

## Objective
Create a virtual machine using the Azure portal and observe how Azure automatically
creates and groups dependent resources inside a resource group.

## Key Concepts Learned
- Azure Resource Groups
- Virtual Machines
- Automatic resource dependency creation
- Resource cleanup to avoid costs

## Outcome
Successfully created a virtual machine and verified all associated resources inside
a single resource group, then deleted the resource group to clean up.
