# Creating an Azure Virtual Machine Using Azure CLI

**Platform:** Microsoft Azure  
**Level:** Azure Fundamentals  


## Objective
Create a Linux virtual machine using the Azure CLI and configure it with a web server
(Nginx) using a Custom Script Extension.

## Tools & Technologies
- Azure Portal
- Azure Cloud Shell
- Azure CLI
- Linux (Ubuntu 22.04)
- Nginx Web Server

## Key Concepts Learned
- Creating resource groups using Azure CLI
- Provisioning Linux virtual machines via CLI
- SSH key-based authentication
- Azure VM Custom Script Extensions
- Post-deployment VM configuration

## Outcome
Successfully deployed a Linux virtual machine using Azure CLI and installed Nginx,
confirming that Azure extensions can be used to automate VM configuration.
