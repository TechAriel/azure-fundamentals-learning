# Configuring Network Access to an Azure Virtual Machine

**Platform:** Microsoft Azure  
**Level:** Azure Fundamentals  


## Objective
Configure inbound network access to an existing Azure virtual machine by modifying
Network Security Group (NSG) rules to allow HTTP traffic on port 80.

## Tools & Technologies
- Azure Portal
- Azure Cloud Shell (Bash)
- Azure CLI
- Network Security Groups (NSGs)
- Linux Virtual Machine
- Nginx Web Server

## Key Concepts Learned
- Public IP addressing
- Network Security Groups (NSGs)
- Inbound security rules
- Rule priority and access control
- Validating network connectivity

## Outcome
Successfully configured an NSG to allow inbound HTTP traffic, enabling public access
to a web server hosted on an Azure virtual machine.
