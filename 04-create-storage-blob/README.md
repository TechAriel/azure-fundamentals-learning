# Create and Configure Azure Blob Storage

**Platform:** Microsoft Azure
**Level:** Azure Fundamentals

## Objective
Creating an Azure Storage Account, uploading a file to Blob container storage, and configyring access permissions to allow public access to the blob.

## Key Concepts Learned
- Azure Storage Accounts
- Blob Containers
- Access levels and anonymous access configuration
- Public vs private blob access
- Storage redundancy (LRS)
- Infrastructure services in Azure

## Outcome
Successfully created a storage account, uploaded a blob, and configured anonymous read access to make the file publicly accessible.
