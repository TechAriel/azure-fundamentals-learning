# Estimating Workload Costs Using Azure Pricing Calculator

**Platform:** Microsoft Azure  
**Level:** Azure Fundamentals  

## Objective
Estimating the monthly cost of migrating a basic internal web application
to Azure using the Azure Pricing Calculator.

## Scenario
A basic ASP.NET web application currently running in a datacenter is being
migrated to Azure. The architecture includes:

- Two Windows Virtual Machines
- A Load Balancer (Application Gateway)
- Azure SQL Database
- ~1 TB monthly network processing
- 730 hours of uptime per month

## Key Concepts Learned
- Cost estimation in Azure
- Compute, database, and networking pricing
- Always-on workload planning
- Regional pricing considerations
- Reviewing and exporting cloud cost estimates
